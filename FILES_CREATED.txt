üöÄ QUICK START
================================================================================

1. Build:
   cd ~/turtlebot3_ws
   colcon build --packages-select turtlebot3_autonav
   source install/setup.bash

2. Launch:
   ros2 launch turtlebot3_autonav full_sim_launch.py

3. Navigate:
   - Use "2D Goal Pose" tool in RViz
   - Click on map to set destination
   - Robot navigates autonomously!

4. Compare:
   python3 scripts/compare_performance.py
